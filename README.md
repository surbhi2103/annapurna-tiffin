# Annapurna Tiffin - Crafted With Mother's Touch 

A modern, high-converting landing page for a premium tiffin service, featuring cinematic scroll-linked animations and a seamless WhatsApp ordering experience.

## 🛠 Tech Stack
- **HTML5 / CSS3** (Vanilla)
- **JavaScript** (Canvas Animation Engine)
- **Tailwind CSS** (via CDN for styling)
- **Google Fonts** (Outfit & Playfair Display)

## 📱 Features
- **Cinematic Hero**: Infinite dolly-in animation sequence.
- **Responsive Navigation**: Glassmorphism navbar with mobile menu.
- **WhatsApp Integration**: Direct-to-order sync with business number.
- **Premium Design**: High-end typography and balanced layout.
